1. Security USB Server Configuration Meeting Minutes

This meeting discussed the setup of a security USB system. Key points included the necessity of security USBs, system setup plans, hardware and software selection, and budget and management strategies. Due to the necessity for security USBs, there is a plan to establish a system, gathering opinions from the infrastructure team. Aiming to commence the project in earnest from next year, we plan to establish two redundant servers and review solutions from various security vendors. A budget of approximately 33 million won has been allocated for hardware, with drafts for the operational server and system software prepared. Furthermore, we plan to verify whether the DB is included in the security USB solution. Discussions were also held about the use of space in the security room and specific technical requirements such as disk capacity. All necessary components must be included in the server, and standard specifications will be presented to receive appropriate proposals from vendors.
